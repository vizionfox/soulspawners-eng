# SoulSpawners ENG

A fully localized, improved version of the SoulSpawners Minecraft plugin. Drops souls only from player mob kills, all messages in English, with `/ss chance` command and fair anti-farm logic.

## Features
- Souls drop ONLY if a player kills a living, spawnable mob (no farm abuse)
- `/ss chance` command to show true in-game drop chance
- Configurable drop rates, recipes, item
- All user-facing messages in English
- Based on the original SoulSpawners (MIT License) by mvk, re-localized and improved by VizionFox

## Installation
1. Put the built JAR in your server's `plugins` folder.
2. Edit `config.yml` to your liking: drop chance, item, recipe.
3. Restart the server.
